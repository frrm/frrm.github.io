Folgende Programme haben wir mehr oder weniger ausführlich vorgestellt:

    • zsh Liebhaber - Die zsh Seite schlecht hin
    • schulas Config - Die Konfiguration die für den Vortrag verwendet wurde

  • bb - Ascii art foo
  • screen - der terminal multiplexer schlecht hin
  • socat - Das schweizer Taschenmesser fürs Netzwerk (netcat++)
    • Folien für socat
  • ipython - enhanced interactive Python shell

Die oben genannten Programme sind alle bis auf bb bei grml, der besten Live CD für Konsolen User, dabei.
